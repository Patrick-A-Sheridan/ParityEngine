A Solo-Developed Full-Stack Symbolic Math & Graphing Engine

MathEngine is a high-performance mathematical playground built to bridge the gap between abstract algebraic manipulation and real-time visual rendering. Unlike simple calculators that rely on eval(), MathEngine utilizes a custom-built Recursive Descent Parser and an Abstract Syntax Tree (AST) to understand and simplify expressions into executable logic trees.

🧠 The Symbolic Brain

  • Context-Aware Tokenization: Differentiates unary vs. binary operators.
  • AST Parsing: Generates hierarchical logic trees for deep analysis.
  • Implicit Multiplication: Handles 2x(x+5) syntax automatically.

📉 Visual Precision

  • Coordinate Mapping: Real-time math-to-pixel transformation.
  • Gap Detection: Smart handling of asymptotes (like 1/x).
  • Dynamic Scaling: Adaptive grid labels based on zoom levels.

Algebraic Simplification

The engine performs multiple reduction passes to simplify expressions before evaluation:

  • Identity: x * 1 → x | x + 0 → x
  • Combining: x + x → 2x | 2x + 3x → 5x
  • Cancellation: x / (x * 5) → 0.2
